## 📌 Sobre o Projeto

O **app-northwind** é o projeto prático da formação **QA Engineer com Playwright** da [Gotas de Tecnologia](https://github.com/cursosgotasdetecnologia).

Desenvolvido do zero ao longo de 94 aulas, cobre o ciclo completo de automação de testes — da escrita dos primeiros locators até a integração com CI/CD no GitHub Actions.

A aplicação testada é uma plataforma de gestão de produtos (Admin View) com funcionalidades de CRUD, filtros dinâmicos, combos e autenticação.

> 📖 Documento vivo — atualizado continuamente conforme a evolução do projeto.

## ⚙️ Como rodar localmente

### Pré-requisitos
- Node.js 20+
- Git

### Instalação

```bash
# Clonar o repositório
git clone git@github.com:cursosgotasdetecnologia/app-northwind.git
cd app-northwind

# Instalar dependências
npm install

# Instalar browser
npx playwright install chromium
```

### Variáveis de ambiente

Criar o arquivo `.env` na raiz:

```env
USER_EMAIL=seu-email@exemplo.com
USER_PASSWORD=sua-senha
BASE_URL=https://url-da-aplicacao.com
```

### Executar

```bash
# Todos os testes
npx playwright test

# Interface visual
npx playwright test --ui

# Spec específico
npx playwright test tests/createProductModal.spec.js

# Relatório HTML
npx playwright show-report

# Allure Report
npx allure generate allure-results && npx allure open
```

## 🔄 CI/CD

Pipeline automático a cada **push** e **pull request** para a `main`.

O Playwright Report é gerado como **artifact** e fica disponível para download direto na aba **Actions** — sem precisar rodar nada localmente para investigar falhas.

```yaml
on:
  push:
    branches: [main]
  pull_request:
    branches: [main]
```
