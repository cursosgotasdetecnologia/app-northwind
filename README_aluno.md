## 📌 Sobre o Projeto

O **app-northwind** é um projeto de estudo profissional focado na **automação de testes End-to-End (E2E)** utilizando **Playwright com JavaScript puro**.

O objetivo principal é aplicar boas práticas de:

- Estruturação de projetos de automação
- Escrita de testes E2E escaláveis
- Versionamento com Git
- Integração Contínua (CI/CD) com GitHub Actions
- Documentação técnica clara e evolutiva

> 📖 Este é um **documento vivo**, que será continuamente atualizado conforme a evolução dos testes e melhorias na arquitetura do projeto.

---

## 🛠️ Tecnologias Utilizadas

| Ferramenta | Finalidade |
|------------|------------|
| **Playwright** | Framework para automação de testes E2E |
| **JavaScript (ES6+)** | Linguagem principal dos testes |
| **Node.js** | Ambiente de execução |
| **Git** | Controle de versão |
| **GitHub (SSH)** | Repositório remoto e colaboração |
| **GitHub Actions** | Pipeline de CI/CD para execução automática dos testes |
